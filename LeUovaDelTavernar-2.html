☁️ Configura il database cloud

Per salvare i dati nel cloud (così non li perdi mai), collega l'app a Firebase di Google — gratuito per sempre per uso familiare. Segui questi passi una sola volta:

  1. Vai su console.firebase.google.com e accedi con il tuo account Google
  2. Clicca "Aggiungi progetto", dai un nome (es. uova-tavernar), disabilita Google Analytics e crea
  3. Nel menu a sinistra, clicca "Firestore Database" → "Crea database" → scegli Modalità test → regione europe-west
  4. Torna alla home del progetto, clicca l'icona </> (Web), dai un nome all'app e registra
  5. Copia i valori dalla schermata firebaseConfig nei campi qui sotto:
const firebaseConfig = {
  apiKey: "AIza...",
  authDomain: "il-tuo-progetto.firebaseapp.com",
  projectId: "il-tuo-progetto",
  storageBucket: "il-tuo-progetto.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123:web:abc123"
};
